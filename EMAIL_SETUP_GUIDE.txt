# STEP-BY-STEP: Enable Automatic Emails

## METHOD 1: Gmail (Easiest - FREE)

### Step 1: Get Gmail App Password
1. Go to: https://myaccount.google.com/apppasswords
2. Login with your Gmail account
3. Click "Select app" → Choose "Mail"
4. Click "Select device" → Choose "Other (Custom name)"
5. Type: EventEngine
6. Click "Generate"
7. Copy the 16-character password (like: abcd efgh ijkl mnop)

### Step 2: Update .env file
Open the .env file and add these lines:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=abcd efgh ijkl mnop
EMAIL_FROM=your-email@gmail.com

Replace:
- your-email@gmail.com with YOUR Gmail address
- abcd efgh ijkl mnop with the password from Step 1

### Step 3: Restart API Server
1. Close the terminal running python run.py
2. Run: python run.py again

### Step 4: Test
1. Register for an event as participant
2. Check your email inbox
3. You should receive confirmation email with QR code!

## METHOD 2: SendGrid (Production Ready - FREE for 100/day)

### Step 1: Sign up
1. Go to: https://sendgrid.com/
2. Create free account
3. Verify email

### Step 2: Get API Key
1. Go to Settings → API Keys
2. Create API Key
3. Copy the key (starts with SG.)

### Step 3: Update .env
SENDGRID_API_KEY=SG.your-key-here
EMAIL_FROM=noreply@yourdomain.com

## TROUBLESHOOTING

If Gmail doesn't work:
1. Enable "Less secure app access" in Google account
2. Or use App Password (recommended)
3. Make sure 2-Factor Authentication is enabled

Test email setup:
python -c "from notifications.email import get_email_service; print(get_email_service().is_configured())"

Should print: True

---

## WHAT EMAILS WILL BE SENT AUTOMATICALLY:

1. Registration Confirmation
   - When participant registers
   - Includes event details
   - Includes QR code for attendance

2. Reminder Emails (if agent is running)
   - Sent before event starts
   - Reminds to confirm attendance

3. Event Updates
   - When event details change
   - When event is cancelled
