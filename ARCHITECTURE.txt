║  REGISTRATION ISSUE SOLUTION                                              ║
╚════════════════════════════════════════════════════════════════════════════╝

Problem: "Unable to open register in participant login"

Root Cause: Event is NOT in "REGISTRATION_OPEN" state

Flow:
    Participant Login → Available Events Tab → See event card
                                                      │
                                                      ▼
                              ┌─────────────────────────────────────┐
                              │  If event.state == REGISTRATION_OPEN│
                              │  → Show "Register Now" button       │
                              │                                     │
                              │  Else (CREATED, SCHEDULED, etc)     │
                              │  → Show "Registration Closed"       │
                              └─────────────────────────────────────┘

Solution:
    1. Login as ORGANIZER
    2. Find your event
    3. Click "Transition..." dropdown
    4. Select "Open Registration"
    5. Event state → REGISTRATION_OPEN
    6. Now participants see "Register Now" button!
